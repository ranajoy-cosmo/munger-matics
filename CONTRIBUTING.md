# Contributing

Full contribution guidelines are available in the [documentation site](https://ranajoy-cosmo.github.io/munger-matics/contributing/).

## Quick Setup

```bash
# Install all dependencies
make install

# Install pre-commit hooks
make hooks
```

## Quick Reference

| Command | Description |
|---|---|
| `make run` | Run the Streamlit dashboard |
| `make test` | Run tests with coverage report |
| `make lint` | Check code style |
| `make format` | Auto-format code |
| `make hooks` | Install pre-commit hooks |
| `make docs` | Serve documentation locally |

For branching conventions, commit format, PR process and code style see the [docs](https://ranajoy-cosmo.github.io/munger-matics/contributing/).
